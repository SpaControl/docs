This documents the EXP1 or EXPAND1 port on the SV and SV Mini controllers

| Pin | Controller | Wifi Adapter        |
|-----|------------|---------------------|
| 1   | 12V        | 12V                 |
| 2   | GND        | GND                 |
| 3   |            |                     |
| 4   |            |                     |
| 5   | TX         | RX (PIN45 on wifly) |
| 6   | RX         | TX (PIN46 on wifly) |
| 7   | GND        | GND                 |
| 8   | 12V        | 12V                 |
